WHAT I BUILT
------------

1. INTERNAL PRODUCT CATALOG PIPELINE
   Problem: Manual product onboarding is slow, descriptions are inconsistent, products take time to go live

   Solution: Automated product catalog system that:
   - Automatically generates high-quality product descriptions using LLM
   - Updates product collections intelligently
   - Integrates with Algolia for enhanced searchability and filtering
   - Ensures products go live on website instantly upon submission
   - No manual review needed (reduces onboarding time by 90%)

   How It Works:
   - Operations team uploads product data (CSV, JSON)
   - LLM generates compelling product description
   - System auto-categorizes product using product attributes
   - Updates Algolia index for search
   - Product instantly appears on website
   - Real-time inventory sync

   Tech Stack:
   - Python backend
   - LLM (Vertex AI / GPT) for description generation
   - Algolia for search indexing
   - FastAPI for APIs
   - MongoDB for product data
   - Scheduled jobs for continuous updates


2. LLM-POWERED AUTOMATION TOOLS FOR INTERNAL WORKFLOWS
   Problem: Operations team spends hours on repetitive manual tasks

   Solution: Built suite of LLM automation tools that:
   - Streamline internal business processes
   - Reduce manual data entry
   - Automate report generation
   - Speed up decision-making
   - Enhance operational efficiency

   Tools Built:
   - Email auto-categorization and response drafting

   Tech Stack:

3. COMPREHENSIVE COMPANY VETTING TOOL
   Problem: Operations team needs to verify if companies/sellers are legitimate before onboarding

   Solution: Built AI-powered digital footprint mapper that:
   - Aggregates company data from across the internet
   - Maps company's digital presence
   - Analyzes publicly available information
   - Identifies potential risks and red flags
   - Generates legitimacy score
   - Assists in fraud prevention

   What It Analyzes:
   - Official website presence and quality
   - Social media accounts (verification status, followers)
   - Business registration data
   - News mentions and press coverage
   - Customer reviews and ratings
   - Domain registration details
   - LinkedIn company profile
   - Google search results analysis
   - Public records and certifications

   How It Works:
   - Input company name/website
   - System scrapes 20+ public data sources
   - LLM analyzes aggregated data
   - Generates comprehensive report with:
     * Digital footprint overview
     * Legitimacy score (0-100)
     * Risk assessment
     * Recommendations (approve, review, reject)

   Tech Stack:
   - Web scraping (Playwright, BeautifulSoup)
   - Google Search API
   - LinkedIn API
   - LangChain for analysis
   - Vertex AI for report generation
   - MongoDB for storing assessments
